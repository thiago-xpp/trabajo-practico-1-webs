# trabajo-practico-1-webs
1- Use h1 para darle un titulo a la web, para que se sepa de que se trata la pagina y h2 para los subtitulos, para que sepan de que se trata el contenido de esa parte/ separar los temas.
2- Se usa ul para hacer una lista sin ordenar, seria para hacer una lista sin que importe en el orden q se escriba, como por ejemplo lista de cosas q te gusta hacer, lista de juegos q jugas, etc.
El ol se usa para hecer una lista ordenada, seria q se usa para hacer una lista donde el orden si importe como se escriba, como por ejemplo pasos para instalar algo, recetas de cocina, instrucciones, etc.
3- El atributo alt es importante ya que sin el no se sabria si la imagen carga o no carga pq no aparecia nada.
4- Mi imagen es de ruta relativa, y elegi esa pq es una imagen muy comun de la portada de Minecraft (del tema de mi pagina).
5- La diferencia entre strong y em, es que stron hace que la palabra/oracion que pongas adentro se convierta en negrita, en cambio em es para que lo que pongas adentro se ponga en cursiva.
